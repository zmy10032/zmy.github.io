北太天元学习28——正态分布

正态分布（又叫高斯分布）是最常见的连续分布函数之一。其概率密度函数如下：
[f(x) = 1/(sqrt(2*\pi)*\sigma)exp(-(x-\mu)^2/(2\sigma^2))，\mu是分布的期望，\sigma是标准差]
标准差越大，图像越矮。图像代码如下：

由于 f(x) 的原函数没有表达式，所以算累积分布函数[P(a <= x <= b) = \int_a^bf(x)dx]相当麻烦。除了特殊的a, b值，一般不能直接求解

过去，统计学家会把[X - N(\mu, \sigma)] 的正态分布转化成[Z - N(0, 1)]
具体来说，就是引入了随机变量[Z = (X - \mu)/\sigma]
统计学家使用 Z表 ，该表根据一系列不同的 a 值给出了
[P(X <= a) = \int_{-\infty}^af(x)dx]
的值

使用北太天元，我们可以绕过标准化的步骤。
在 examples/normcdf.m 函数中，可以直接计算任意正态分布的累积分布函数

我们可以用 randn 生成服从 N(0, 1) 的随机数
[要生成服从N(\mu, \sigma)的随机数，可以用 \mu + randn * \sigma]

北太天元学习29-等待时间和指数分布

如果一个随机变量 X 的概率密度函数为
[
f(x) =
\begin{cases}
\lambda e^{-\lambda x} & \text{ } x > 0,
\
0 & \text{ } x \le 0.
\end{cases}
]
那么 X 满足 指数分布，记作[X \sim Exponential(\lambda)]
其累积分布函数 (cdf) 为
[F(x) = P(X \le x) = \int_{-\infty}^{x} f(x) dx = 1 - e^{-\lambda x}]

提问：
我们用 rand 生成均匀分布的随机数，
用 randn 生成正态分布的随机数，
那怎么生成指数分布的随机数呢 ?

  1. 生成（0， 1）的随机数 rand
  2. 由 F(x) = r 算出 [x = -log(1-rand)/\lambda \text{（北太天元里log是自然对数，log10(10) = 1）}]

x 就是符合指数分布的随机数
代码如下：

n = 1000  % 实验次数
a = 2  % 参数
x = zeros(n, 1);
for i = 1:n
    x(i) = -log(1-rand)/a;
end
histogram(x, 20);
% xlabel(生成的数);
% ylabel(生成的概率);
title('生成指数分布随机数');

我们用一个例子研究 指数分布 与 泊松分布 的关系：
    假设一个灯泡的平均寿命是100个小时，我们想知道1000个小时里我们将使用多少个灯泡(假设灯泡熄灭后会立即更换)
    因为1000/100 = 10，所以泊松分布的速率为10。我们进行2000次模拟：

北太天元学习22-醉汉回家

在上p中，我们用到了这个函数：

function move = move_rand(r)
move = sign(r - 0.5);
end

先只看 x 轴
很显然，当我们测试总次数为奇数时，醉汉不会在偶数处停下
总次数为偶数时，不会再奇数处停下

所以我们要稍稍改进一下
代码如下：

function move = move_rand(r)
move = zeros(size(r))

ind = find(r < 1/3)
if ~isempty(ind)
move(ind) = -1
end

ind = find(r >= 1/3 && r < 2/3)
if ~isempty(ind)
move(ind) = 0
end

ind = find(r > 2/3)
if ~isempty(ind)
move(ind) = 1
end
end

  • find(condition) 可以找到满足条件的索引
  • isempty(object) 可以判断对象是否为空

北太天元学习20——多准则决策(Multi-criteria decision making)方法

每个人做决策时往往要考虑很多因素

宋小煜同学是一名高中毕业生，想在无意识心理学和玄学中选一个作为自己的专业。他要考虑三个因素：个人喜好(LOVE)、发展空间(SPACE)和就业前景(MONEY)
宋小煜同学面临的就是一个“既要，又要，还要”的多准则问题
解决这类问题，我们经常采用层次分析法，和上p的谷歌搜索有亿点像
让我们来问宋小煜同学，如果 love 的重要性是 1 的话，space 的重要性是几？这是个选择题，可以选 非常不重要、不重要、一丢丢不重要、一样重要、一丢丢重要、重要、非常重要。分别用 1/7、1/5、1/3、1、3、5、7 表示
分别设 love、space、money 为 1 ，我们得到下面的矩阵：
              love space money
love          1      1/3     1/7
space       3       1       1/3
money      7        3         1
观察这个矩阵，发现每列的和不是 1 ，so 不能把它看成一个状态转移矩阵。但每行的含义更明确，所以我们可以把这个矩阵转置一下：
A = A.'
完事之后，我们以第一列[1; 1/3; 1/7]为例，这个就和上p的含义很像了，表示 love 对其他因素的作用。我们用
A(:, 1) = A(:, 1)/norm(A(:, 1))
对这个向量进行“归一化”，也就是把它变成单位向量[21/31; 7/31; 3/31]
对其他列我们也做一样的操作，得到一个新矩阵（我觉得就是状态转移矩阵）
然后我们做和上p一样的丝滑小连招：

  1. 找最大的特征值对应的特征向量
  2. 把它化成单位向量
  3. 把每个元素取个绝对值（负的肯定不对劲嘛）

就能找到 love、space、money 在宋小煜同学心中的重要性了。
这个例题的答案是[0.68, 0.23, 0.09]，我们记作 a
然后我们问宋小煜同学 无意识心理学 和 玄学 对 love 的重要性，具体流程和前面一样（加入玄学是1……）
得到新的向量 blove ：[0.23; 0.77]
同理可得 bspace 和 bmoney

最后就能得到这两个学科的总作用了:
无意识心理学：a * [blove(1); bspace(1), bmoney(1)]
玄学：a * [blove(2); bspace(2), bmoney(2)]

北太天元学习24-保费定价和二项分布

二项分布用于模拟多次实验的情况，每个实验都有两种可能性：成功或失败

小栗子：
投掷一枚硬币，让 X_i 表示第 i 次实验出现的结果，1/0 表示成功/失败。导出一个新的随机变量 X = X_1 + X_2 + ……
那么 X 的取值可以为0，1，2，3，……, n。
X = k 的概率是 P(X = k) = C_n^k * p^k * (1-p)^(n-k)
其中 C_n^k 表示从 n 个不同的元素里选 k 个的组合数
我们说 X 服从二项分布，记作 X ~ B(n, k)

小例题：
某保险公司推出意外伤害险，每位参保人交付保险费，出险时获得4万元保险费。已知一年中一个人的出现率为0.3%，有1000个人参加保险，要让公司获利10万元以上的概率超过90%，问公司至少要向每位参保人收多少参保费
思路：
每人交 x 元，假设一年中有 y 个人申请理赔，要让公司获利10万元的概率高于90%，则
P(1000 * x - 40000 * y >= 100000) > 0.9
即
P((y <= 1000 * x - 100000)/40000) > 0.9
我们可以算出 y 的二项分布，找到 P(y <= k) = 0.9 对应的 k 值就能算出 x 了

代码如下：

  • factorial(n) 得到 n 的阶乘
  • find(cumsum(vector) <= 0.9) 得到 vector 中的元素从前往后求和不大于0.9 的 索引

北太天元学习17-特征向量和特征值

视频前面讲了 特征值和特征向量 ，在《线性代数》里也学过了

  • eig(A) 求矩阵 A 的特征值（eig 是 eigenvalue 特征值 的缩写）
  • [V, D] = eig(A)可以让 A 的 特征向量 组成 V 的列，A 的 特征值 组成 D 主对角线上的元素
  • sum(vector) 能算出向量上所有元素的和
  • norm(vector) 能算出向量的“二范数”，个人理解就是向量的长度。北太天元算出来的特征向量的二范数都是1
  • n x n 对角矩阵的特征值就是对角线上的数，特征向量就是 n 阶单位矩阵的列向量

北太天元学习18-离散模型

以人口模型为例

设 x_{n} 是时间 n 时的人口数量，则递归关系可以由以下方程给出：
x_{n} = f (x_{0}, x_{1}, x_{2}, ......, x_{n-1})
也就是说，x_{n} 是先前所有时间人口的函数。

  • 形式为 x_{n} = f (x_{n-1}) 称作一阶递推关系，因为它只依赖于上一个时间的值；
    x_{n} = f (x_{n-1}, x_{n-2}) 是二阶递推关系

    如果一个模型用一个或多个递推关系来描述某些现象，则称其为离散动力系统

  • 假设人口在下一时间的增长与当前人口成正比，则
    x_{n+1} - x_{n} = k*x_{n} ，即x_{n+1} = (k+1)*x_{n}
    这是一个一阶递推关系，被称为指数增长模型，也叫马尔萨斯模型

  • 通常情况下，人口不会指数增长，增长到一定程度会减缓。逻辑斯蒂克模型刻画了增长率随人口接近最大容量的变化：
    x_{n+1} - x_{n} = r * x_{n} * (1 - x_{n}/K)
    其中K是承载能力。
    化成二次函数就是：x_{n+1} = (1 + r) * x_{n} - r * x_{n}^2/K
    这是一个非线性递推关系

  • 对于指数增长模型，根据 x_{0} 和 x_{1} ，可以确定 k

  • 对于逻辑斯蒂克模型，根据 x_{0}, x_{1}, x_{2}, 可以确定 r 和 K

现在引入一个新词语 —— 态

  • 我们仅仅考虑生活在草原上的两种动物：兔子和狼 构成的系统

  • 这个系统在时间 t 的态可以用向量表示：x_{t} = [ 兔_{t}, 狼_{t} ]

  • 下面给出一个简单的 捕食者-被捕食者（predator_pray） 离散模型
    兔_{t + 1} = (1 + r) * 兔_{t} - f * 狼_{t} * 兔_{t}
    狼_{t + 1} = (1 + w) * 狼_{t} + g * 狼_{t} * 兔_{t}
    r, w 是兔子和狼的自然增长率，f 和 g 反应兔狼相逢对他们数量的影响。这些参数可以由 [兔_{0}; 狼_{0}], [兔_{1}; 狼_{1}] 来确定。
    下面给出脚本代码：

    x = [1000, 1];
    r = 0.01; f = 1e-4; w = 2e-3; g = 3e-5;
    para = [r; w; w; g];
    x1 = pp_step(x, para);
    x2 = pp_step(x1, para);
    A = [x(1), 0, -x(1)*x(2), 0;
    0, x(2), 0, x(1)*x(2);
    x1(1), 0, -x1(1)*x1(2), 0;
    0, x1(2), 0, x1(1)*x1(2)];
    b = [x1 - x; x2 - x1];
    para = A\b;
    N = 100;
    p = zeros(2, N+1);
    p(:, 1) = x;
    for i = 1:x
    p(:, n+1) = pp_step(p(:, n), para);
    end
    subplot(1, 2, 1)
    plot(0:N, p(1, :), ‘r-‘, ‘LineWidth’, 3)
    title(“兔子”);
    subplot(1, 2, 2)
    plot(0:N, p(2, :), ‘b-‘, ‘LineWidth’, 3)
    title(“狼”);
    function y = ppstep(x, para)
    [r, w, f, g] = deal(para(1), para(2), para(3), para(4));
    y(1) = (1 + r) * x(1) - f * x(1) * x(2)
    y(2) = (1 + w) * x(1) + g * x(1) * x(2);
    end

  • pp_step() 是一个子函数

  • deal() 可以把一个数列中变量的值分发给另外的变量

  • A 矩阵是 ==r, w, f, g 的系数矩阵==
