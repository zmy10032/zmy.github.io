北太天元学习19-谷歌搜索算法(page rank algorithm)简介

我们假设有4个网页
在初始时刻，网页1有很多人在浏览，其他网页无人浏览
所有浏览网页的人记作1
每隔一个单位时间都有部分人转移到别的网页
定义在时间n时的态为
p_n = [p_{1, n}, p_{2, n}, p_{3, n}, p_{4, n}]
其中p_{i, n}表示在n时刻处在网页i上的浏览者的比例
所以初始态p_0 = [1, 0, 0, 0]
p_{n + 1} 和 p_{n} 之间的关系是建模的重点：
p_{n + 1} = A * p_{n}
其中矩阵 A 称为状态转移矩阵
则p_{1, n+1} = A(1, 1) * p_{1, n} + A(1, 2) * p_{2, n} + A(1, 3) * p_{3, n} + A(1, 4) * p_{4, n}
如果考虑有人不想浏览网页了，可以增加第五个“网页”，到第五个“网页”就表示下网了。

我们可以这样来建模：

  1. 如果第i个网页有k个其他网页的链接，那么转移到它们的概率是1/(k + 1)
  2. 如果该网页没有其他网页的链接，那么浏览者会随机选择一个网站

给一个具体的例子：
网页1拥有链接2、4，网页2没有链接，网页3有链接1，网页4有链接2、3
那么 A =
1/3 1/4 1/2 0
1/3 1/4 0 1/3
0 1/4 1/2 1/3
1/3 1/4 0 1/3
x_0 = [1; 0; 0; 0]

脚本代码如下：

A = [1/3 1/4 1/2 0
1/3 1/4 0 1/3
0 1/4 1/2 1/3
1/3 1/4 0 1/3];
初始p = [1; 0; 0; 0];
N = 20;
p = zeros(4, N);
p(:, 1) = 初始p;
for n = 1:N
p(:, n + 1) = A * p(:, n);
end
hold on
for j = 1:4
plot(1:N, p(j, :), ‘LineWidth’, 3);
str(j) = sprintf(“网页%d”, j);
label_str(j) = sprintf(“网页%d最终占比%d%%”, j, 100*p(:, end));
end
hold off;
legend(str);
figure(2)
pie(p(:, end), label_str);

  • str(j) = sprintf(......) 将输出写入到字符串中，而不是直接输出到屏幕
  • figure(2) 创建编号为 2 的图形窗口。如果 2 号窗口已经存在，它将霸占这个窗口
  • pie(p(:end), label_str) 画一个饼图，把label_str里的字符串按顺序放在饼各部分旁边

视频最后有个发现：最终态是稳定的，并且这个态 ppp 和 A 最大特征值对应的特征向量 pp 成比例:
ppp = pp/norm(pp, 1)

  • norm(pp, 1) 得到 pp 的长度

但原理是啥并没有解释

北太天元学习14——消元法解线性方程组

举个栗子

A =
1 2 3 5
4 5 6 2
7 8 9 0

A =
1  2  3  5
0 -3 -6 -18
0  0  0  1

A =
1  0 -1 -7
0  1  2  6
0  0  0  1

不难发现，步骤就是

  1. 把主对角线下面的元素变成 0
  2. 把主对角线上的元素变成 1 的同时把主对角线上面的元素变成 0

下面是代码：

function C = simpMatrixEx(A)
if nargin ~= 1
error(“请输入一个矩阵”);
end
format rat;
B = A;
[m, n] = size(B);
for i = 1:m-1
for j = i+1:m
if B(i, i) ~= 0
B(j, :) = B(j, :) - B(i, :)/B(i, i)*B(j, i);
end
end
end
C = B;
for j = m:-1:2
if B(j, j) ~= 0
B(j, :) = B(j, :)/B(j, j);
end
for i = j-1:-1:1
if B(j, j) ~= 0
B(i, :) = B(i, :) - B(j, :)*B(i, j);
end
end
end
end

北太天元学习13——调试m函数

和 Ideal 几乎一样

  1. 设置断点：
    点一下行数左边就行，或者用dbstop指令。例如，dbstop("isprimeEx", 5) 就是在 isprimeEx 文件的第5行设置断点，==注意 isprimeEx 后面不要加 .m==
  2. 开启调试：
    可以按“调试”按钮或者用 dbon 开始调试
    在 m 函数中，可以在K >> 的后面输入要传入的参数，例如：K >> isprime(666)
  3. 下一步：
    可以按“单步调试”或者用dbnext执行下一步
    按“继续调试”或者用dbcontinue是执行到下一个断点
  4. 检查值：
    把鼠标悬停在某变量上方可以看到变量的值，也能从工作区直接看
  5. 修改值：
    在工作区 双击变量 或者 右键变量再按编辑值 可以实时修改变量的值
    ==注意不要在调试过程中修改 m 文件的代码，不然会有意外惊喜的哟==
  6. 结束调试：
    直接按停止按钮或者用dboff结束调试
  7. 清除断点：
    一个一个在脚本上手动清除或者用dbclear一键清除

北太天元学习11——不动点迭代求一元函数f(x)的近似根

不动点迭代：
对于函数 f(x) ，如果 f(x) = x 成立，那么就称 x 为函数 f(x) 的不动点
我们可以用不动点迭代来求一些方程的根

  • 例子
    求 x^3^ - 5x - 2 = 0 的根

    1. 画出函数图像：

    x = -1:0.1:1;
    fh = @(t) t.^3 - 5*t - 2;
    % 这里的 fh 是匿名函数的名字，@(t) 表示接收一个 t 作为参数（记住就行）。因为 t 有可能是矢量，所以 t^3^ 用 t.^3 来表示。
    plot(x, fh(x), ‘LineWidth’, 5, ‘Color’, ‘Red’);
    2. 从图中看出来函数的一个根在 -1 到 1 之间
    3. 把 x^3^ - 5x - 2 = 0 化成 x = (x^3^ - 2)/5 ，就能构造不动点迭代
    x_{n+1} = (x_{n}^3 - 2)/5
    如果 x_{n} 的极限存在，那么这个极限就是 (x^3^ - 2)/5 的不动点，也是原方程的根
    4. 迭代的停机理论：让两步迭代的差小于一个**忍量(tolerance)**，例如|x_{n+1} - x_{n}| < 1e-4

    下面是具体实现：

    x = -0.6; % 初始化 x
    gh = @(t) (t.^3 - 2)/5 % 定义匿名函数
    tol = 1e-4; % 设置忍量
    n = 1; % 设置索引
    x(n+1) = gh(x(n));
    while abs(x(n+1) - x(n)) > tol & 直到停机终止运行
        n = n + 1; % 更新索引
        x(n+1) = gh(x(n)); % 得到下一个点
    end
    x(end) % 输出迭代 x 最后的值

    我们还可以把迭代过程画出来：

    hold on
    plot(0:x.length-1, x, ‘r-‘, ‘LineWidth’, 3);
    % ‘r-‘ 中的 “r” 是红色，“-” 的意思是把点连起来
    sh = scatter(0:x.length-1, x, ‘filled’);
    set(sh, ‘SizeData’, 500);
    title([‘不动点迭代 x(end) = ‘, num2str(x(end))]);
    hold off

北太天元学习10-如何写一个m函数

啸知识点：

  • n = input("...") 运行时在屏幕上显示…的内容，并可以输入一个值，用 n 来接收
  • break 可以停止这次循环，并开始下一次循环（while, for循环等等）
  • ctrl + Q 可以退出运行程序
  • 文件命名规则：不能用数字开头，但中文可以 :)
  • nargin 是一个 “系统变量”，等于传入的参数数量
  • return 可以终止 m 函数的运行
  • error("...") 可以终止函数运行并且在命令行窗口用红字显示…的内容，即自定义报错信息（再也不用看奇奇怪怪的报错啦！）
  • mod(x, y) 和 rem(x, y) 有区别吗？
    1. x 和 y 同号时没区别
    2. x 和 y 异号时，mod(x, y) 符号与 y 相同，rem(x, y) 符号与 x 相同
      例如：mod(5, -3) = -1，rem(5, -3) = 2

问题：判断数字n是否是素数

  • 以前写的脚本模式

    while 1
      n = input(“请输入一个正整数：”);
        if n > 0 && mod(n, 1) == 1
          break;
        end
    end
    if (n == 1)
      素数判断结果 = false
    elseif (n == 2 || n == 3)
      素数判断结果 = true
    else
      % 计算n的平方根
      sqrt_n = sqrt(n);
      % 检测 n 是否能被大于1且小于等于 sqrt_n 的数整除
      for m = 2:sqrt_n
       if mod(n, m) == 0
         素数判断结果 = false
       end
    end
    if (素数判断结果 == true)
      fprintf(“%d 是素数”, n);

    但我们希望把判断一个数p是否是素数的算法重用，也就是要将p作为参数传入，然后函数计算出结果

  • m 函数可以满足这个要求
    步骤：

    1. 建立一个文件，叫isPrimeEx()，==注意文件名和函数名要一样==;（prime：质数，ex：example）
    2. 把算法放入function和end圈定的范围内，格式为：

      function 返回值 = 函数名(参数)
        代码
      end

    下面把以前写的脚本改成m函数：

    function result = isPrimeEx(p)
      if nargin ~= 1
        error(“请输入一个正整数”);
      end
      while 1
        n = input(“请输入一个正整数：”);
          if n > 0 && mod(n, 1) == 1
            break;
          end
    end
      if (n == 1)
        素数判断结果 = false;
        return;
      elseif (n == 2 || n == 3)
        素数判断结果 = true;
        return;
      else
        % 计算n的平方根
        sqrt_n = sqrt(n);
        % 检测 n 是否能被大于1且小于等于 sqrt_n 的数整除
        for m = 2:sqrt_n
         if mod(n, m) == 0
           素数判断结果 = false;
            return;
         end
      end
    end

    其实就是加了个头和尾嘛
    当然，m 函数的格式还可以是

    function [输出结果列表] = 函数名(输入参数列表)
      巴拉巴拉；
    end

    例如：

    function [m, n] = mEx(x, y)
      m = x + y;
      n = x - y;
      return;
    end
    % 可以这么用：
    [res1, res2] = mEx(2, 3);
    % 如果不用列表来接收，比如只写一个 mEx(2, 3)，那 mEx(2, 3) 就只返回 5

==视频结尾：会这些就差不多毕业了！==

绘制一元函数图像 —— 北太天元学习4

原理

x, y 是两个相同维数的行向量，x 中的每个元素对应一个横坐标，y 中的每个元素对应一个纵坐标，二者确定一个点。用plot(x, y)将这些点用直线连起来，得到图像。

  • 举例：
    x = [1, 2, 3]
    y = [3, 2, 1]
    则 plot(x, y) 表示将(1, 3)、 (2, 2)、 (3, 1)三点连起来形成的图像。

分得越细，越逼近真实曲线

  • 举例：
    x = linspace(-1, 1, 100(越大越逼近))
    y = x.^2
    plot(x, y)

取名

  • xlabel('x')给x轴取名
  • ylabel('y')给y轴取名
  • title('北太天元绘制 y=x∧2 x=-1,...1')给整个图取名
