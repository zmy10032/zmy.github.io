北太天元学习16-矩阵运算

视频前面讲了 矩阵乘法、逆矩阵 等等在《线性代数》里学过的东西，这里就略去了

  • diag(1, 5, 6) 可以生成对角矩阵
    1 0 0
    0 5 0
    0 0 6
  • inv(A) 可以求 A 的逆矩阵
  • 矩阵计算的时候可能会有误差，比如 1e-16 其实就是 0
  • 已知 Ax = b ，就能用 x = inv(A)*b 或者 x = A\b 来算 x 了（“\”是左除）
  • det(A) 得到矩阵的行列式
  • A = rand(m, n) 生成一个 m x n 的随机矩阵，A = rand(k) 生成一个 k x k 的随机矩阵
