北太天元学习29-等待时间和指数分布

如果一个随机变量 X 的概率密度函数为
[
f(x) =
\begin{cases}
\lambda e^{-\lambda x} & \text{ } x > 0,
\
0 & \text{ } x \le 0.
\end{cases}
]
那么 X 满足 指数分布，记作[X \sim Exponential(\lambda)]
其累积分布函数 (cdf) 为
[F(x) = P(X \le x) = \int_{-\infty}^{x} f(x) dx = 1 - e^{-\lambda x}]

提问：
我们用 rand 生成均匀分布的随机数，
用 randn 生成正态分布的随机数，
那怎么生成指数分布的随机数呢 ?

  1. 生成（0， 1）的随机数 rand
  2. 由 F(x) = r 算出 [x = -log(1-rand)/\lambda \text{（北太天元里log是自然对数，log10(10) = 1）}]

x 就是符合指数分布的随机数
代码如下：

n = 1000  % 实验次数
a = 2  % 参数
x = zeros(n, 1);
for i = 1:n
    x(i) = -log(1-rand)/a;
end
histogram(x, 20);
% xlabel(生成的数);
% ylabel(生成的概率);
title('生成指数分布随机数');
