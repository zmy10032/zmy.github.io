北太天元学习9–绘制散点图

一些小指令

  • clc 清屏
  • clear all 清除工作区
  • clf 清除图里面的内容
  • close all 关掉所有的图

上面这几个一起用就能保证界面干净

画图技巧

x = 0:0.01:2pi;
y_sinx = sin(x);
y_cosx = cos(x);
x_point = 0:pi/2:2pi;
正弦_point = sin(x_point);
余弦_point = cos(x_point);

hold on % 保证前一个图不会被后一个图冲掉，即在页面上保留多张图
plot(x, y_sinx, “LineWidth”, 2.0); % “LineWidth 可以设置线宽”
plot(x, y_cosx, “LineWidth”, 4.0);
sh1 = scatter(x_point, 正弦_point, “filled”); % 用scatter 来画散点，”filled 表示颜色填充”
sh2 = scatter(x_point, 余弦_point);
set(sh1, ‘SizeData’, 500); % SizeData 控制散点大小
set(sh2, ‘SizeData’, 1000);
set(sh2, ‘LineWidth’, 7);
% 设置散点sh2边缘颜色(rgb值)、sh1填充颜色、sh2形状
set(sh2, ‘MarkerEdgeColor’, [1, 0, 0]);
set(sh2, ‘MarkerFaceColor’, [0, 0.5, 0.9]);
set(sh2, “Marker”, ‘+’);
legend(“sinx”, “cosx”, “sin点”, “cos点”, ‘FontSize’, 18); % 取名
hold off; % 与 hold on 相配
