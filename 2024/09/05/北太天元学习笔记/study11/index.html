北太天元学习11——不动点迭代求一元函数f(x)的近似根

不动点迭代：
对于函数 f(x) ，如果 f(x) = x 成立，那么就称 x 为函数 f(x) 的不动点
我们可以用不动点迭代来求一些方程的根

  • 例子
    求 x^3^ - 5x - 2 = 0 的根

    1. 画出函数图像：

    x = -1:0.1:1;
    fh = @(t) t.^3 - 5*t - 2;
    % 这里的 fh 是匿名函数的名字，@(t) 表示接收一个 t 作为参数（记住就行）。因为 t 有可能是矢量，所以 t^3^ 用 t.^3 来表示。
    plot(x, fh(x), ‘LineWidth’, 5, ‘Color’, ‘Red’);
    2. 从图中看出来函数的一个根在 -1 到 1 之间
    3. 把 x^3^ - 5x - 2 = 0 化成 x = (x^3^ - 2)/5 ，就能构造不动点迭代
    x_{n+1} = (x_{n}^3 - 2)/5
    如果 x_{n} 的极限存在，那么这个极限就是 (x^3^ - 2)/5 的不动点，也是原方程的根
    4. 迭代的停机理论：让两步迭代的差小于一个**忍量(tolerance)**，例如|x_{n+1} - x_{n}| < 1e-4

    下面是具体实现：

    x = -0.6; % 初始化 x
    gh = @(t) (t.^3 - 2)/5 % 定义匿名函数
    tol = 1e-4; % 设置忍量
    n = 1; % 设置索引
    x(n+1) = gh(x(n));
    while abs(x(n+1) - x(n)) > tol & 直到停机终止运行
        n = n + 1; % 更新索引
        x(n+1) = gh(x(n)); % 得到下一个点
    end
    x(end) % 输出迭代 x 最后的值

    我们还可以把迭代过程画出来：

    hold on
    plot(0:x.length-1, x, ‘r-‘, ‘LineWidth’, 3);
    % ‘r-‘ 中的 “r” 是红色，“-” 的意思是把点连起来
    sh = scatter(0:x.length-1, x, ‘filled’);
    set(sh, ‘SizeData’, 500);
    title([‘不动点迭代 x(end) = ‘, num2str(x(end))]);
    hold off
