北太天元学习22-醉汉回家

在上p中，我们用到了这个函数：

function move = move_rand(r)
move = sign(r - 0.5);
end

先只看 x 轴
很显然，当我们测试总次数为奇数时，醉汉不会在偶数处停下
总次数为偶数时，不会再奇数处停下

所以我们要稍稍改进一下
代码如下：

function move = move_rand(r)
move = zeros(size(r))

ind = find(r < 1/3)
if ~isempty(ind)
move(ind) = -1
end

ind = find(r >= 1/3 && r < 2/3)
if ~isempty(ind)
move(ind) = 0
end

ind = find(r > 2/3)
if ~isempty(ind)
move(ind) = 1
end
end

  • find(condition) 可以找到满足条件的索引
  • isempty(object) 可以判断对象是否为空
