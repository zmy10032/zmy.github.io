北太天元学习21——随机模型

  • rand 生成一个（0，1）的随机数
    要生成一个在2~8的随机数，用 2*rand + 8 即可
  • rand(n) 生成一个 n x n 的随机矩阵，每个元素都是（0，1）
  • rand(m, n) 生成一个 m x n 的随机矩阵

下面是 醉汉走丢 的例子：
一个醉汉在每个单位时间里在 x，y 方向都要走 +1 或者 -1 个长度
下面是画出醉汉可能路径的脚本：

load_plugin(“time”)
N = 100; % 每一次出发走的总步数
总次数 = 200;
x(1) = 0;
y(1) = 0;
for m = 1:总次数
hold on
plot(-25:25, -25:25);
for n = 1:N
sx = sign(rand - 0.5);
x(n + 1) = x(n) + sx;
sy = sign(rand - 0.5);
y(n + 1) = y(n) + sy;
plot(x(n:n+1), y(n:n+1), ‘LineWidth’, 10);
str = sprintf(“第%d次出发，第%d步”, m, n);
title(str);
pause(0.2);
end
clf;
hold on
plot(-25:25, -25:25);
plot(x, y, ‘LineWidth’, 10)
pause(1);
hold off
end

  • load_plugin("time") 加载的插件用于后面的 pause(114514) ，就是让画图时停顿114514秒，方便看清
  • clf 可以清除画面内容
  • plot(-25:25, -25:25); 控制画面大小为 50 x 50
