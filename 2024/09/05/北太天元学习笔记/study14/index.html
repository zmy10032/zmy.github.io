北太天元学习14——消元法解线性方程组

举个栗子

A =
1 2 3 5
4 5 6 2
7 8 9 0

A =
1  2  3  5
0 -3 -6 -18
0  0  0  1

A =
1  0 -1 -7
0  1  2  6
0  0  0  1

不难发现，步骤就是

  1. 把主对角线下面的元素变成 0
  2. 把主对角线上的元素变成 1 的同时把主对角线上面的元素变成 0

下面是代码：

function C = simpMatrixEx(A)
if nargin ~= 1
error(“请输入一个矩阵”);
end
format rat;
B = A;
[m, n] = size(B);
for i = 1:m-1
for j = i+1:m
if B(i, i) ~= 0
B(j, :) = B(j, :) - B(i, :)/B(i, i)*B(j, i);
end
end
end
C = B;
for j = m:-1:2
if B(j, j) ~= 0
B(j, :) = B(j, :)/B(j, j);
end
for i = j-1:-1:1
if B(j, j) ~= 0
B(i, :) = B(i, :) - B(j, :)*B(i, j);
end
end
end
end
