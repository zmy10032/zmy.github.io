for循环 – 北太天元学习5

for i= 开始数:步增数:结束数
  程序语句1
  程序语句2
    …
end

  • 例1：计算6！
    p = 1;
    for i = 1:6
    p = p * i
    end

  • 例2：计算 1/1^2 + 1/2^2 + … + 1/50^2

    1. s = 0;
      for i = 1:50
      s = s + 1/i^2
      end
    2. x = 1:50
      sum(1/x^2)

    方法ii不使用for循环而直接计算1/1^2 + 1/2^2 + … + 1/50^2 的和, 会计算得更快一些。
