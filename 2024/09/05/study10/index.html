北太天元学习10-如何写一个m函数

啸知识点：

  • n = input("...") 运行时在屏幕上显示…的内容，并可以输入一个值，用 n 来接收
  • break 可以停止这次循环，并开始下一次循环（while, for循环等等）
  • ctrl + Q 可以退出运行程序
  • 文件命名规则：不能用数字开头，但中文可以 :)
  • nargin 是一个 “系统变量”，等于传入的参数数量
  • return 可以终止 m 函数的运行
  • error("...") 可以终止函数运行并且在命令行窗口用红字显示…的内容，即自定义报错信息（再也不用看奇奇怪怪的报错啦！）
  • mod(x, y) 和 rem(x, y) 有区别吗？
    1. x 和 y 同号时没区别
    2. x 和 y 异号时，mod(x, y) 符号与 y 相同，rem(x, y) 符号与 x 相同
      例如：mod(5, -3) = -1，rem(5, -3) = 2

问题：判断数字n是否是素数

  • 以前写的脚本模式

    while 1
      n = input(“请输入一个正整数：”);
        if n > 0 && mod(n, 1) == 1
          break;
        end
    end
    if (n == 1)
      素数判断结果 = false
    elseif (n == 2 || n == 3)
      素数判断结果 = true
    else
      % 计算n的平方根
      sqrt_n = sqrt(n);
      % 检测 n 是否能被大于1且小于等于 sqrt_n 的数整除
      for m = 2:sqrt_n
       if mod(n, m) == 0
         素数判断结果 = false
       end
    end
    if (素数判断结果 == true)
      fprintf(“%d 是素数”, n);

    但我们希望把判断一个数p是否是素数的算法重用，也就是要将p作为参数传入，然后函数计算出结果

  • m 函数可以满足这个要求
    步骤：

    1. 建立一个文件，叫isPrimeEx()，==注意文件名和函数名要一样==;（prime：质数，ex：example）
    2. 把算法放入function和end圈定的范围内，格式为：

      function 返回值 = 函数名(参数)
        代码
      end

    下面把以前写的脚本改成m函数：

    function result = isPrimeEx(p)
      if nargin ~= 1
        error(“请输入一个正整数”);
      end
      while 1
        n = input(“请输入一个正整数：”);
          if n > 0 && mod(n, 1) == 1
            break;
          end
    end
      if (n == 1)
        素数判断结果 = false;
        return;
      elseif (n == 2 || n == 3)
        素数判断结果 = true;
        return;
      else
        % 计算n的平方根
        sqrt_n = sqrt(n);
        % 检测 n 是否能被大于1且小于等于 sqrt_n 的数整除
        for m = 2:sqrt_n
         if mod(n, m) == 0
           素数判断结果 = false;
            return;
         end
      end
    end

    其实就是加了个头和尾嘛
    当然，m 函数的格式还可以是

    function [输出结果列表] = 函数名(输入参数列表)
      巴拉巴拉；
    end

    例如：

    function [m, n] = mEx(x, y)
      m = x + y;
      n = x - y;
      return;
    end
    % 可以这么用：
    [res1, res2] = mEx(2, 3);
    % 如果不用列表来接收，比如只写一个 mEx(2, 3)，那 mEx(2, 3) 就只返回 5

==视频结尾：会这些就差不多毕业了！==
